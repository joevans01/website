Visual Basic is an IDE (Integrated Development Environment) which is easy to pick up, this is the reason why it is taught at GCSE level and used in examinations. The latest and most commonly used version is the 2010 express edition. It is commonly used becasue its features are compatible with the most amount of operating systems, these are: Win XP, 7, 8.1 and 10.

Task 1 - Outputting Messages

The first task is going to show you how to output what is entered in a textbox.

    1.Open VisualBasic 2010
    2.Create a new project with a name
    3.Drag a textbox and a button from the side panel called the toolbox

    4.When the design has been made, each of the items need to be named. This can be done in the properties which are by the bottom of the screen by default.
    5.After I have to name each of the items appropriately we can move onto the coding part of the project. To start double click the button and you will be created with a page like so:
    6.Next we need to start coding. Firstly, I am going to declare a variable that will store what is in the textbox. To do this I will type 'Dim' and then type a word that will store the value. I am going to call it 'text'.
    7.Then I am going to use a messagebox to display the text. To do this I will use 'messagebox.show' and then inside brackets put the name of the variable which is 'text'.
    8.Then you need to press F5 to test the program. When the form loads type anything in the textbox then press the button to see if a message showing the text appears.
